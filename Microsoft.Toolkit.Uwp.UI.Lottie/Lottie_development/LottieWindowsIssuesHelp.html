LV0003

Issues that may be reported by the Lottie Viewer

Code
Description
Details
LP0001
Failed to parse JSON
The file was not recognized as valid JSON.
LP0002
Fatal error: Unexpected field: fieldName
The parser encountered a field that it was not expecting. Parsing was halted.
LP0003
Chars field
The composition contains a "chars" section. This is currently an unsupported feature.
LP0004
Fonts field
The composition contains a "fonts" section. This is currently an unsupported feature.
LP0005
Unsupported asset type: assetType
The composition contains an asset type that is not recognized by the parser. The asset will be ignored.
LP0006
Layer with render = false
A layer had its render flag set to false. This is currently an unsupported feature.
LP0007
Illustrator layers
The composition contains Illustrator layers. These layers are not supported.
LP0008
Layer effects
The composition contains layer effects. These effects are not supported.
LP0009
Mattes
The composition contains mattes. Mattes are not yet supported.
LP0010
Masks
The composition contains masks. Masks are not yet supported.
LP0011
Time remapping of precomp layers
The composition has precomp layers that have their times remapped. Time remapping is not yet supported.
LP0012
Unexpected shape content type: type
The parser encounted an unexpected shape content type. The shape content will be ignored.
LP0013
Gradient strokes
The composition uses gradient strokes. Gradient strokes are not yet supported.
LP0014
Polystar property animation
A polystar has an animated property. Animation of some polystar properties is not yet supported.
LP0015
Expressions
The composition uses expression. Expressions are not supported.
LP0016
Ignored field: parsingPhase.fieldName
The parser encountered a field that it does not know how to handle.
LT0001
Rectangle with animated size and trim path
A rectangle shape has both its size and trim path animated. This is currently not supported.
LT0002
Animated trim offset with static trim offset
A trim path has an animated offset and a non animated offset. This is currently not supported.
LT0003
Multiplication of two or more animated values
2 or more values needed to be multiplied, e.g. opacity and color, but they are both animated. This is currently not supported.
LT0004
Blend mode mode
Not all blend modes are yet supported.
LT0005
Combining animated shapes
Combining of 2 more shapes is not currently supported if they are animated.
LT0006
Gradient fill
Gradient fills are not yet supported.
LT0007
Gradient stroke
Gradient strokes are not yet supported.
LT0008
Image assets
Image assets are not supported.
LT0009
Image layers
Image layers are not supported.
LT0010
Merging a large number of shapes
There is a limit to how many shapes can be merged.
LT0011
Multiple animated rounded corners
A shape layer can have at most 1 animated rounded corner.
LT0012
Multiple fills
A shape layer can have at most 1 fill.
LT0013
Multiple strokes
A shape layer can have at most 1 stroke.
LT0014
Multiple trim paths
A shape layer can have at most 1 trim path.
LT0015
Opacity and color animated at the same time
Either opacity or color can be animated, but not both. This limitation exists because opacity animation is implemented as animation of the alpha channel of a color.
LT0016
Path with rounded corners
Rounded corners on paths are not supported.
LT0017
Polystar
Polystars are not supported.
LT0018
Repeater
Repeaters are not supported.
LT0019
Text layer
Text layers are not yet supported.
LT0020
3d composition
3d composition is not supported.
LT0021
3d layer
3d layers are not supported.
LT0022
Time stretch
Time stretch is not yet supported.
LV0001
Layer name has in-point after out-point
In-points must be before out-points.
LV0002
Layer name is in a cycle
A chain of layer references forms a cycle.
LV0003
Invalid layer parent id
A layer has a parent that does not exist.
